# PMS 데이터 구조 명세서 (DATA SCHEMA GUIDE)

이 문서는 범용 PMS 뼈대가 정상적으로 작동하기 위해 필요한 **필수 데이터 구조**를 정의합니다. 사용자는 자신의 엑셀 파일 헤더를 아래의 '시스템 변수명'에 매핑하여 시스템을 즉시 구동할 수 있습니다.

## 1. 필수 데이터 항목 (Core Metrics)

시스템 대시보드와 리포트를 생성하기 위해 시트(`raw data` 탭)에 반드시 포함되어야 하는 항목입니다.

| 시스템 변수명(Key) | 필수 여부 | 데이터 타입 | 설명 | 추천 엑셀 헤더명 |
| :--- | :---: | :--- | :--- | :--- |
| `date` | **필수** | Date/String | 생산 일자 (YYYY-MM-DD) | 날짜, 생산일, 전표일자 |
| `final` | **필수** | Number | 최종 합격 생산 수량 | 최종생산량, 합격수량, 양품수 |
| `defect` | **필수** | Number | 총 불량 발생 수량 | 총불량, 불량수, NG수 |
| `month` | 선택 | Number | 해당 월 (1~12) | 월, 생산월 |
| `weekNum` | 선택 | String | 주차 (예: 2024-W15) | 주차, 작업주 |

---

## 2. 공정별 실적 항목 (Process Metrics)

공정별 생산량을 트래킹하고 싶을 때 추가하는 항목입니다. (설정 시 차트에 자동 반영)

| 시스템 변수명(Key) | 데이터 타입 | 설명 | 예시 |
| :--- | :--- | :--- | :--- |
| `step1` | Number | 1단계 공정 생산량 | 성형, 가공, 투입 |
| `step2` | Number | 2단계 공정 생산량 | 조립, 용접, 도장 |
| `step3` | Number | 3단계 공정 생산량 | 포장, 검사, 출하 |

---

## 3. 상세 불량 항목 (Defect Details)

불량 유형별 파레토 차트를 그리기 위해 필요한 항목입니다.

| 시스템 변수명(Key) | 데이터 타입 | 설명 | 예시 |
| :--- | :--- | :--- | :--- |
| `d1` | Number | 불량 유형 1 수량 | 찌그러짐, 스크레치 |
| `d2` | Number | 불량 유형 2 수량 | 오염, 치수미달 |
| `d3` | Number | 불량 유형 3 수량 | 조립누락, 기타 |

---

## 4. 환경 변수 설정 방법 (Mapping)

엑셀 파일의 헤더가 위와 다를 경우, Google Apps Script의 **[프로젝트 설정] > [스크립트 속성]**에 다음과 같이 JSON 형식으로 입력합니다.

**Key**: `COLUMN_MAPPING`
**Value**:
```json
{
  "date": "작업일자",
  "final": "양품수합계",
  "defect": "불량코드합계",
  "step1": "성형공정",
  "d1": "치수불량",
  "d2": "표면불량"
}
```

이렇게 설정하면 시스템이 자동으로 엑셀의 "작업일자" 컬럼을 읽어 `date` 데이터로 처리합니다.
